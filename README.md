# Note me

Daily Note App  
url : [https://chingu-project-noteme.vercel.app/](https://chingu-project-noteme.vercel.app/)

## Instructions:

**To install the dependencies**:

```cmd
npm install
```

**To check the style if it follows the style guide**:

```cmd
npm run lint
```

**To start the server in development**:

```cmd
npm run dev
```

**To build it**:

```cmd
npm run build
```
